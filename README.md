# 📝 Attendance Application (Absentap)

A web-based application built using **Laravel 10 / 11** for attendance management. This repository contains a comprehensive guide on the installation steps and local development environment configuration using **Laragon**.

---

## 🚀 System Prerequisites

Before starting the installation, ensure you have the following software installed on your computer:

* **Laragon** (Latest version recommended with PHP >= 8.1)
* **Composer** (Usually bundled within Laragon)
* **Git**
* **Project folder must already be placed inside Laragon's www folder**

---

## 🛠️ Installation Steps via Laragon

Follow this step-by-step guide to run the application on your local machine:

### 1. Place the Project in the `www` Folder
Ensure this project folder is located inside your Laragon root directory, typically found at:
`C:\laragon\www\absentap`

### 2. Start Laragon Services
1. Open the **Laragon** application.
2. Click the **Start All** button to turn on Apache/Nginx and MySQL.

### 3. Install PHP Dependencies
Open the terminal (you can click the **Terminal** button in Laragon) inside your project folder, then run the following command:
```bash
composer install

### 4. Running Project
Right click on Laragon, open the www folder and select projects.
